# ProyectoSemestralBackEnd
Proyecto semestral "SearchSport" para la salida intermedia de informática
Descripción: Aplicación web y móvil que permite a los usuarios localizar centros deportivos en un mapa, verificar la disponibilidad de canchas de fútbol (inicialmente) en tiempo real y realizar la reserva y pago de forma digital.
Instrucciones de instalación (Java version 17, base de datos: por definir).
